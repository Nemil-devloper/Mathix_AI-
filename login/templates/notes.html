Math Notes

1. Derivatives

The derivative of a function measures how the function value changes as its input changes. For example, the derivative of f(x) = x² is f'(x) = 2x.

2. Integrals

Integrals are used to find areas under curves. The integral of f(x) = x is ∫f(x)dx = (1/2)x² + C, where C is the constant of integration.

3. Trigonometric Identities

Key identities include sin²(x) + cos²(x) = 1 and tan(x) = sin(x)/cos(x).

4. Quadratic Equations

A quadratic equation is in the form ax² + bx + c = 0. Solutions can be found using the quadratic formula: x = (-b ± √(b² - 4ac)) / (2a).

Previous Year Questions

1. What is the derivative of f(x) = 3x² + 5x - 7?

Answer: f'(x) = 6x + 5

2. Calculate the integral ∫(2x + 3)dx.

Answer: (x² + 3x + C)

3. Solve the quadratic equation x² - 5x + 6 = 0.

Answer: x = 2, x = 3

4. Prove the identity: sin²(x) + cos²(x) = 1.

Answer: This follows from the Pythagorean theorem.
